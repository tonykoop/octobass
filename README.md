# Octobass

Status: L1 concept packet

Authority: this repository is a V5 concept study for private review. It is not
fabrication authority, not a measured acoustic packet, and not a tuning plan.
All build-critical dimensions, forces, string data, linkage geometry, body
structure, and stop positions are pending measurement or expert review.

The octobass concept is a giant bowed, three-string sub-contrabass at roughly
the prompt-supplied 3.5 m scale. Because a player cannot stop the strings by
hand at that size, the core design problem is a playable string-stopping
mechanism: hand levers and foot pedals actuate tangent, clamp, or padded-finger
stops against the strings while the player bows the instrument.
