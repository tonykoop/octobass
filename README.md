# Octobass

Status: L2 concept / pending-measurement packet

Authority: this repository is a V5 concept study for private review. It is not
fabrication authority, not a measured acoustic packet, and not a tuning plan.
All build-critical dimensions, forces, string data, linkage geometry, body
structure, and stop positions are pending measurement or expert review.

The octobass concept is a giant bowed, three-string sub-contrabass study.
Because a player cannot practically stop the strings by hand at that scale, the
core design problem is a playable string-stopping mechanism: hand levers and
foot pedals actuate tangent, clamp, or padded-finger stops against the strings
while the player bows the instrument. The L2 uplift adds subsystem planning,
candidate part classes, and promotion gates without asserting measured scale,
tuning, force, or linkage values.

## Known L2 Boundaries

- No CAD, DXF, or measured drawings exist in this packet.
- No tuning, string gauges, scale length, body outline, plate thickness, force,
  or lever-ratio values are claimed.
- The three-string layout remains a concept family constraint, not a measured
  stringing plan.
- Any future promotion must start with reference research, ergonomic mockups,
  linkage force tests, and a string/stop measurement plan.
